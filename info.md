# Grow Room Manager

Manage medical cannabis flowering rooms using the Athena Pro Line cultivation methodology.

## Features

- **Multi-Room Support** - Manage F1, F2, F3+ rooms
- **Grow Journaling** - Save notes and camera snapshots
- **Automated Tasks** - 84-day Athena schedule with calendar & todo integration
- **Status Sensors** - Track day, phase, EC, and dryback targets

## Quick Start

1. Add configuration to `configuration.yaml`
2. Create calendar and todo entities for each room
3. Set start date and generate tasks
4. Follow the Athena Pro Line schedule!

See [full documentation](https://github.com/goatboynz/HA-Grow-Assist) for details.
